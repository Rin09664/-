Давайте поговорим о чем этот проект, для какого окружения оно будет полезно и какую проблему решает.

 Цель: Безопасная передержка с онлайн-камерами и ветеренарной поддержкой.
 Пользователи: Владельцы и проверенные сиделки.
 
•  Функции:
  •  Поиск с фильтрами (камера, вет. поддержка).
  •  Онлайн-наблюдение.
  •  Чат с сиделкой/ветеринаром.
  •  Безопасные платежи.
  •  Преимущества: Прозрачность, безопасность, решение проблемы "черной передержки".

  На каких языках программирования можно сделать такое приложение и какие мы получим привилегии взяв именно эти языки.

  Для этого приложения я рассматривала множество языков и мой выбор пал на Python и Java
  
  Причина моего выбора ;
  
  Python (в начале):

•  Быстро запустить: Легко учить, много готовых штук.
•  Проверить идею: Сделать работающее приложение быстро и дешево.
•  "Умные" функции: Много готовых решений для ИИ (рекомендации, анализ видео).

Java (потом, если нужно):

•  Много пользователей: Выдержит большую нагрузку, если приложение станет популярным.
•  Безопасность: Защитит данные и деньги пользователей.
•  Надежность: Будет стабильно работать, даже если много людей одновременно используют приложение.

Поисковая вкладка ; панель управления ; камера :

<img width="747" height="807" alt="image" src="https://github.com/user-attachments/assets/23c68cca-5142-468b-8a57-614875c47823" />

На первой картинке мы наблюдаем все функции. Внизу мы видим панель управления. Поиск - поисковая функция, которая помогает вам найти сиделку под вашего питомца. Избранное - ваши объявлении которые вам понравились. Объявления - для нянечек , чтобы размещать свои услуги. Сообщения - связь с сиделкой/хозяином. Профиль - ваш профиль, ФИО и контактный номер. Сверху в правом окне мы наблюдаем черную точку, она для камер. Нажав на нее, у вас потребуют логин и пароль. Надо ввести туда данные которые даст вам нянечка. Благодаря этому вы сможете наблюдать со своим животным в любое время и бть спокойным.
